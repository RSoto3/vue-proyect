# Nombre del Proyecto

## Tecnologías Utilizadas

* Vue.js
* Vuetify
* Pinia
* ApexCharts

## Instrucciones

1.  **Instalar dependencias:**

    ```bash
    npm install
    ```

2.  **Ejecutar el servidor:**

    ```bash
    npm run dev
    ```
